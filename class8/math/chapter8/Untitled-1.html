CHAPTER 8 — EXERCISE 8.1

1. Add the following.
(i) Add: \(ab - bc,\; bc - ca,\; ca - ab\)

\[ (ab - bc) + (bc - ca) + (ca - ab) \]

\[ = ab - bc + bc - ca + ca - ab \]

\[ = (ab - ab) + (-bc + bc) + (-ca + ca) \]

\[ = 0 + 0 + 0 = 0 \]

Final answer: \(0\)
(ii) Add: \(a - b + ab,\; b - c + bc,\; c - a + ac\)

\[ (a - b + ab) + (b - c + bc) + (c - a + ac) \]

\[ = a - b + ab + b - c + bc + c - a + ac \]

\[ = (a - a) + (-b + b) + (-c + c) + ab + bc + ac \]

\[ = 0 + 0 + 0 + ab + bc + ac \]

\[ = ab + bc + ac \]

Final answer: \(ab + bc + ac\)
(iii) Add: \(2p^{2}q^{2} - 3pq + 4\) and \(5 + 7pq - 3p^{2}q^{2}\)

\[ (2p^2q^2 - 3pq + 4) + (5 + 7pq - 3p^2q^2) \]

\[ = 2p^2q^2 - 3pq + 4 + 5 + 7pq - 3p^2q^2 \]

\[ = (2p^2q^2 - 3p^2q^2) + (-3pq + 7pq) + (4 + 5) \]

\[ = -p^2q^2 + 4pq + 9 \]

Final answer: \(-\,p^{2}q^{2} + 4pq + 9\)
(iv) Add: \(l^{2}+m^{2},\; m^{2}+n^{2},\; n^{2}+l^{2},\; 2lm+2mn+2nl\)

\[ (l^2 + m^2) + (m^2 + n^2) + (n^2 + l^2) + (2lm + 2mn + 2nl) \]

\[ = l^2 + m^2 + m^2 + n^2 + n^2 + l^2 + 2lm + 2mn + 2nl \]

\[ = (l^2 + l^2) + (m^2 + m^2) + (n^2 + n^2) + 2lm + 2mn + 2nl \]

\[ = 2l^2 + 2m^2 + 2n^2 + 2lm + 2mn + 2nl \]

\[ = 2(l^2 + m^2 + n^2 + lm + mn + nl) \]

Final answer: \(2(l^{2}+m^{2}+n^{2}+lm+mn+nl)\)
2. (a) Subtract \(4a - 7ab + 3b + 12\) from \(12a - 9ab + 5b - 3\).

Set up subtraction:

\((12a - 9ab + 5b - 3) - (4a - 7ab + 3b + 12)\)

Remove parentheses (change signs of second bracket):

\[ = 12a - 9ab + 5b - 3 - 4a + 7ab - 3b - 12 \]

Group like terms:

\[ = (12a - 4a) + (-9ab + 7ab) + (5b - 3b) + (-3 - 12) \]

\[ = 8a - 2ab + 2b - 15 \]

Final answer: \(8a - 2ab + 2b - 15\)
2. (b) Subtract \(3xy + 5yz - 7zx\) from \(5xy - 2yz - 2zx + 10xyz\).

Set up subtraction:

\((5xy - 2yz - 2zx + 10xyz) - (3xy + 5yz - 7zx)\)

Remove parentheses and change signs of second bracket:

\(= 5xy - 2yz - 2zx + 10xyz - 3xy - 5yz + 7zx\)

Group like terms:

\(= (5xy - 3xy) + (-2yz - 5yz) + (-2zx + 7zx) + 10xyz\)

\(= 2xy - 7yz + 5zx + 10xyz\)

Final answer: \(2xy - 7yz + 5zx + 10xyz\)
2. (c) Subtract \(4p^{2}q - 3pq + 5pq^{2} - 8p + 7q - 10\) from \(18 - 3p - 11q + 5pq - 2pq^{2} + 5p^{2}q\).

Write subtraction:

\((18 - 3p - 11q + 5pq - 2pq^{2} + 5p^{2}q) - (4p^{2}q - 3pq + 5pq^{2} - 8p + 7q - 10)\)

Remove brackets and change sign of second bracket:

\(= 18 - 3p - 11q + 5pq - 2pq^{2} + 5p^{2}q - 4p^{2}q + 3pq - 5pq^{2} + 8p - 7q + 10\)

Combine like terms:

\(= (5p^{2}q - 4p^{2}q) + (5pq + 3pq) + (-2pq^{2} - 5pq^{2}) + (-3p + 8p) + (-11q - 7q) + (18 + 10)\)

\(= p^{2}q + 8pq - 7pq^{2} + 5p - 18q + 28\)

Final answer: \(p^{2}q + 8pq - 7pq^{2} + 5p - 18q + 28\)
