EXERCISE 3.2 SOLUTIONS

1. Find \( x \) in the following figures.

Figure (a)

Given:

Three exterior angles \( 125^\circ \), \( 125^\circ \) and \( x \)

We know that:

Sum of the exterior angles = \( 360^\circ \)

\[ 125^\circ + 125^\circ + x = 360^\circ \]

\[ 250^\circ + x = 360^\circ \]

\[ x = 360^\circ - 250^\circ \]

\[ x = 110^\circ \]

Answer for part (a): \( x = 110^\circ \)

Figure (b)

Given:

Here, ABC is a straight line.

By Linear Pair:

\[ \angle 1 + \angle B = 180^\circ \]

\[ \angle 1 + 90^\circ = 180^\circ \]

\[ \angle 1 = 180^\circ - 90^\circ \]

\[ \angle 1 = 90^\circ \]

Thus, our exterior angles are:

\[ x, 90^\circ, 60^\circ, 90^\circ, 70^\circ \]

We know that:

Sum of exterior angles = \( 360^\circ \)

\[ x + 90^\circ + 60^\circ + 90^\circ + 70^\circ = 360^\circ \]

\[ x + 310^\circ = 360^\circ \]

\[ x = 360^\circ - 310^\circ \]

\[ x = 50^\circ \]

Answer for part (b): \( x = 50^\circ \)

2. Find the measure of each exterior angle of a regular polygon of:

(i) 9 sides   (ii) 15 sides

Key Formula:

The measure of each exterior angle of a regular polygon = \( \frac{360^\circ}{n} \)

Where \( n \) is the number of sides.

(i) For a regular polygon with 9 sides:

Each exterior angle = \( \frac{360^\circ}{9} = 40^\circ \)

(ii) For a regular polygon with 15 sides:

Each exterior angle = \( \frac{360^\circ}{15} = 24^\circ \)

3. How many sides does a regular polygon have if the measure of an exterior angle is 24°?

We know that for a regular polygon:

Number of sides = \( \frac{360^\circ}{\text{Exterior angle}} \)

Given exterior angle = 24°

Number of sides = \( \frac{360^\circ}{24^\circ} = 15 \)

The regular polygon has 15 sides.

4. How many sides does a regular polygon have if each of its interior angles is 165°?

Key Relationship:

For any polygon: Interior angle + Exterior angle = 180°

Given interior angle = 165°

So, exterior angle = 180° - 165° = 15°

Number of sides = \( \frac{360^\circ}{\text{Exterior angle}} = \frac{360^\circ}{15^\circ} = 24 \)

The regular polygon has 24 sides.

5 (a). Is it possible to have a regular polygon with measure of each exterior angle as 22°?

For a regular polygon, the number of sides must be a whole number.

Number of sides = \( \frac{360^\circ}{\text{Exterior angle}} = \frac{360^\circ}{22^\circ} \approx 16.36 \)

Since 16.36 is not a whole number, a regular polygon cannot have an exterior angle of 22°.

No, it is not possible to have a regular polygon with each exterior angle measuring 22°.

(b). Can it be an interior angle of a regular polygon? Why?

For a regular polygon, the interior angle must satisfy:

\( \frac{(n-2) \times 180^\circ}{n} = \text{Interior angle} \)

And the exterior angle = \( \frac{360^\circ}{n} \) must be a divisor of 360°.

If the interior angle is given, we can find the exterior angle as 180° - interior angle.

Then \( n = \frac{360^\circ}{\text{Exterior angle}} \) must be a whole number.

An angle can be an interior angle of a regular polygon only if \( \frac{360^\circ}{180^\circ - \text{interior angle}} \) is a whole number.

6(a). What is the minimum interior angle possible for a regular polygon? Why?

The minimum interior angle occurs when the polygon has the maximum number of sides.

As the number of sides increases, the interior angle approaches 180° from below.

But for a regular polygon, the minimum number of sides is 3 (triangle).

For an equilateral triangle, each interior angle = 60°.

The minimum interior angle for a regular polygon is 60° (for an equilateral triangle).

(b). What is the maximum exterior angle possible for a regular polygon?

The maximum exterior angle occurs when the polygon has the minimum number of sides.

The minimum number of sides for a regular polygon is 3 (triangle).

For an equilateral triangle, exterior angle = \( \frac{360^\circ}{3} = 120^\circ \).

The maximum exterior angle for a regular polygon is 120° (for an equilateral triangle).
